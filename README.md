# Nordic Equity Screener

A Python-based quantitative screening tool to identify investment opportunities in the OMX Nordic 40 index.

## Article
This code supports the analysis in ["The Analyst's Filter"](https://theoslobrief.substack.com/) on The Oslo Brief.
